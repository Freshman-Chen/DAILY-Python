### 数据类型：

**数值类型**：整型，浮点型，复数

**文本序列**：字符串

**序列类型**：列表，元组，range

**映射类型**：字典

**集合类型**：集合

| **对象** |              **示例**              |                           **说明**                           |
| :------: | :--------------------------------: | :----------------------------------------------------------: |
|   整型   |              1024、78              |                        整数大小无限制                        |
|  浮点型  |         123.45、23.0、0.78         | 取值范围:2.225 073 858 507 201 4 e-308到1.797 693 134 862 315 7 e+308 |
|  字符串  |    '1024'、"test"、'''10test'''    |             使用成对的单引号、双引号或三引号界定             |
|  range   |      range(1,10,2)、range(2)       | range(start, stop[，step])返回的是从start 到stop，步长为step的整数序列，常用于控制循环 |
|   列表   |   [1,2,1,3,4]、[1,2,'sd',"sda"]    | 用一对中括号界定，元素间用逗号分隔，元素可为数字、字符串、列表、元组等任意数据类型 |
|   元组   |   (1,2,1,2,5)、(1,2,'sa')、(1,)    |   用一对小括号界定，元素间用逗号分隔，元素可为任意数据类型   |
|   集合   | {1,2,3,4,}、{'h','l', "h","hello"} | 用一对大括号界定，元素无顺序，集合中元素具有唯一性，元素间用逗号分隔 |
|   字典   |      {'age':40,'name'='zhao'}      | 用一对大括号界定，元素间用逗号分隔，每个元素包含“键”和“值”两部分 |
